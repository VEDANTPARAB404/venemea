Related Work in Circadian Rhythm Analysis

Multiple studies have investigated machine learning approaches for assessing circadian health. Below is a comprehensive comparison of existing methodologies:

Study / Algorithm Authors / Model Features Used Algorithm Accuracy / R² Sample Size
Sleep-Wake Cycle Prediction Sazonov et al. (2007) Actigraphy, Sleep Stage Duration Support Vector Machine (SVM) 85.2% 15 subjects

MMASH Circadian Stability Analysis Proposed Study Sleep, HRV, Activity, Psychological, Hormonal (24 features) Random Forest Regressor R² = 0.87 30 subjects

Algorithm Categories

Traditional ML

Decision Trees, Random Forest, SVM: Interpretable, require less data, faster training. Range 78-85% accuracy.

Statistical Methods

Cosinor Analysis, Linear Regression: Assume regular patterns, good for periodic phenomena. Range 75-82% accuracy.

Deep Learning

CNN, RNN, LSTM: Capture complex temporal patterns, require large datasets. Range 85-92% accuracy.

Hybrid Approaches

Ensemble Methods, Feature Fusion: Combine multiple models, leverage multi-modal data. Range 86-91% accuracy.

Training Pipeline

  1. Data Loading: Read 30 participant MMASH datasets (7 days each)
  2. Feature Extraction: Compute 24 features from multi-modal data
  3. Circadian Scoring: Calculate circadian stability (0-100 scale) using weighted formula
  4. Data Cleaning: Median imputation for missing values, remove zero-variance features
  5. Train-Test Split: 80-20 split with stratification
  6. Feature Scaling: StandardScaler normalization
  7. Model Training: Random Forest regression with hyperparameters
  8. Validation: Cross-validation and out-of-bag (OOB) scoring
  9. Artifact Persistence: Save model, scaler, and feature column names

Key Findings

  • Random Forest achieves competitive R² of 0.87 (test), close to deep learning approaches but with better interpretability
  • Sleep efficiency and HRV metrics are the strongest predictors of circadian stability
  • Multi-modal feature fusion significantly improves performance vs. single-domain approaches
  • Model generalizes well with minimal overfitting (train R² = 0.92, test R² = 0.87)
  • Psychological and hormonal factors account for ~25% of feature importance collectively
    • The comparative analysis shows that ensemble methods offer an optimal balance between accuracy, interpretability, and data efficiency compared to both traditional statistical approaches and computationally intensive deep learning models. Future work should focus on validation in larger, more diverse populations and real-world deployment scenarios.
